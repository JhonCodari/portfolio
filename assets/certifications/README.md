# Certificações - Imagens

Esta pasta contém as imagens das certificações e cursos.

## Como adicionar imagens

### Certificações AWS
- **aws-cloud-practitioner.png** - Badge da certificação AWS Cloud Practitioner
- **aws-solutions-architect.png** - Badge da certificação AWS Solutions Architect

Baixe os badges oficiais da AWS em: https://www.credly.com/users/[seu-usuario]/badges

### Cursos
- **alura-java.png** - Certificado Alura - Java
- **alura-docker.png** - Certificado Alura - Docker
- **alura-sql.png** - Certificado Alura - SQL
- **alura-angular.png** - Certificado Alura - Angular
- **udemy-microservices.png** - Certificado Udemy - Microservices
- **udemy-aws.png** - Certificado Udemy - AWS
- **dio-spring-boot.png** - Certificado DIO - Spring Boot
- **dio-redis.png** - Certificado DIO - Redis
- **company-architecture.png** - Certificado Empresa - Arquitetura
- **company-security.png** - Certificado Empresa - Segurança

## Formato recomendado
- Formato: PNG com fundo transparente
- Tamanho: 800x600px (ou proporção similar)
- Qualidade: Alta resolução
- Nome: kebab-case (ex: aws-cloud-practitioner.png)
